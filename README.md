Bem-vindo(a) ao meu repositório **User_Interface**! Este espaço foi criado exclusivamente para armazenar, organizar e demonstrar meus projetos de design de interface, estudos visuais e fluxos de experiência do usuário (UX/UI).

---

##  Ferramentas Utilizadas

* **Figma:** Criação de wireframes, layouts de alta fidelidade e protótipos interativos.
* **Plugins:** Utilizados para testes de contraste (acessibilidade), bibliotecas de ícones e placeholders de imagem.
